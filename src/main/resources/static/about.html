WebSSH

实现 网页 中的 ssh 客户端操作，支持多用户多线程操作！！！！支持指定ssh 连接！！！

项目简介

WebSSH 是一个基于 Java 的 WebSSH 项目，使用 Spring Boot 作为服务器，基于 SSH 协议，支持 Linux、Windows 和 Mac 等系统。该项目无需任何外部依赖，性能优异。

使用方法

可以直接通过 docker 来实现快速部署，也可以通过 java -jar 来直接启动 jar 包，下面是 docker 以及 Java操作的命令。

docker pull beardedmanzhao/webssh && docker run -d -p 8080:8080 beardedmanzhao/webssh
java -jar WebSSH.jar
